Available software

All available software

The software on BMRC cluster is maintained using module system.

The list of all available software on the BMRC cluster with a short description can be found here.

To get the list of all available software run:

$ module avail

To search for a specific software in the list, for example, Python versions, enter:

$ module avail 2>&1 | grep -i python
