File system

Files organisation

On the BMRC cluster, there are two folders for each user - home folder and file folder.

Your home folder will be located at /users/<group>/<username> - with <group> and <username> being your group name and username, which is a random combination of three letters and three numbers for data security reasons. The size of this folder is limited to 10GB - you should use it only for storing essential configuration files which software often expects to find there, like the Bash configuration file .bashrc.

Your folder for general files will be located at /well/<group>/users/<username>. Please use this folder to store your code, results of data analysis and other files.

Symbolic links are set up in your home folder for convenience to point to the location in the file folder to organise the files according to their functionality or use, such as code development (devel), data storage (archive, mirror) and data analysis (work)

These are:

devel -> /well/<group>/users/<username>/devel

notebooks -> /well/<group>/users/<username>/notebooks

work -> /well/<group>/users/<username>/work

archive -> /well/<group>/archive

The mirror folder is linked to /well/kir/mirror – the location of shared external database data.

Each group has folders for sharing data within the group, called projects, and between different groups, called shared. They are located at /well/<group>

Storage quota

The storage folder at /well/group/users/<username> shares in your group's allocation for disk space so please use the space responsibly. Compress the files when not in use or for longer term storage.
